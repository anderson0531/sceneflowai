# SceneFlow AI Studio - 5 Minute Feature Storyboard Shot List

This plan is built for a grounded "this is real" product walkthrough.
Runtime target: 5:00 total (10 blocks x 0:30 each).

## Capture Rules (for consistent quality)

- Record in 1440p or 1080p at 30fps.
- Show real cursor movement and click feedback.
- Keep at least one loading/progress state per block for authenticity.
- Avoid hyper-fast cuts; let each primary UI state breathe for 3-5 seconds.
- Keep narration factual and tied to what is visible on screen.
